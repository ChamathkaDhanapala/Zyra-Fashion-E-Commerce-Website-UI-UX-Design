# 👗 Zyra-Fashion-E-Commerce-Website-UI-UX-Design
A modern Fashion E-Commerce Website UI/UX design created in Figma, showcasing a clean and stylish shopping journey. Includes homepage (guest and logged-in views), product listing, product detail, cart, checkout, and profile pages. Focused on delivering an elegant, user-friendly, and conversion-focused fashion shopping experience.

---

## 🎨 Project Overview  

The goal of this design is to create a **clean, stylish, and user-friendly shopping experience** for a fashion brand.  
The project includes both **guest user flow** and **logged-in user flow**, showing how the homepage and features adapt based on context.  

---

## 🚀 Features  

- ✨ **Modern Homepage** with hero banner, collections, and promotions  
- 👗 **Product Browsing** with filters  
- 👠 **Product Detail Page** with gallery, sizes, colors, and details  
- 🛒 **Cart & Checkout Flow** for seamless shopping  
- 🙍‍♀️ **Profile Page** with order history and logout  
- ❓ **FAQ Page** for customer support
